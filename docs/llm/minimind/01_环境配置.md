## 1. 安装python环境
**uv cache 配置**

在C:\Users\ASUS\AppData\Roaming\uv 目录下创建uv.toml配置文件，里面可以做全局uv配置
```toml
# uv 全局配置文件
# Windows 路径: %APPDATA%\uv\uv.toml

# ============================================
# 1. 自定义缓存目录
# ============================================
# 将 uv 的缓存路径修改到 D 盘，避免占用 C 盘空间
cache-dir = "D:/code/uv_cache"

# ============================================
# 2. 清华 PyPI 镜像源（国内加速）
# ============================================
# 设置默认的包下载索引为清华镜像源
[[index]]
url = "https://pypi.tuna.tsinghua.edu.cn/simple/"
default = true

# ============================================
# 3. （可选）备用镜像源 - 阿里云
# ============================================
# 如果需要备用源，可以取消下面的注释
# 注意：设置了 default = true 的源只有一个，其他源会被视为额外索引
# [[index]]
# url = "https://mirrors.aliyun.com/pypi/simple/"
```

**问题：在windows环境下，默认会安装cpu版本torch，使用uv安装：**
```
uv pip install torch torchvision --index-url https://download.pytorch.org/whl/cu130
```

**安装剩余的包**
```
uv pip install -r requirements.txt
```

## 2.下载模型和数据集
首先需要配置下载模型的环境变量，将模型cache下载到机械硬盘上。
HF_HOME
MODELSCOPE_CACHE_DIR

下载命令：
```
uv run modelscope download --dataset gongjy/minimind_dataset README.md
```
可以看到，readme文件下载到了以下路径
`E:\cache\modelscope_cache\datasets\gongjy\minimind_dataset`

使用modelscope下载剩余的数据集
```
uv run modelscope download --dataset gongjy/minimind_dataset pretrain_t2t_mini.jsonl sft_t2t_mini.jsonl
```

下载模型
```
uv run modelscope download --model gongjy/minimind-3
```

运行测试（使用预先下载好的模型参数）
```
python eval_llm.py --load_from minimind-3
```
但是上面这个写法有一点问题，因为你是使用的modelscope下载的模型，保存的模型参数存到了modelscope的缓存路径，而使用transformer库加载模型时，默认是去hugging face的缓存路径去找模型，因此肯定会找不到需要重新下载，所以这里给他指定模型路径。
```
uv run eval_llm.py --load_from E:\cache\modelscope_cache\models\gongjy\minimind-3
```

原因是可以看一下源代码加载模型的逻辑，这里他写了一个如果发现加载路径中包含model，那么使用手工实现的torch模型去读取参数权重，而不是使用transformer库中的from_pretrain函数。因为我们的路径中包含了model，所以他使用了加载torch模型的分支，由于当前还没有做预训练，所以没有得到模型参数，因此报错正常。只需将判断逻辑修改以下即可。

```
def init_model(args):
    tokenizer = AutoTokenizer.from_pretrained(args.load_from)
    if 'model' in args.load_from and 'modelscope_cache' not in args.load_from:
        model = MiniMindForCausalLM(MiniMindConfig(
            hidden_size=args.hidden_size,
            num_hidden_layers=args.num_hidden_layers,
            use_moe=bool(args.use_moe),
            inference_rope_scaling=args.inference_rope_scaling
        ))
        moe_suffix = '_moe' if args.use_moe else ''
        ckp = f'./{args.save_dir}/{args.weight}_{args.hidden_size}{moe_suffix}.pth'
        model.load_state_dict(torch.load(ckp, map_location=args.device), strict=True)
        if args.lora_weight != 'None':
            apply_lora(model)
            load_lora(model, f'./{args.save_dir}/{args.lora_weight}_{args.hidden_size}.pth')
    else:
        model = AutoModelForCausalLM.from_pretrained(args.load_from, trust_remote_code=True)
    get_model_params(model, model.config)
    return model.half().eval().to(args.device), tokenizer
```
